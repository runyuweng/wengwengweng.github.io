关于js中的this

网上this的指向很多都是说指向引用他的对象，
其实this的指向因根据不同情况而定。

一、在函数中调用

首先思考为啥下面这块代码的输出是什么。

这个代码块的输出又是什么。

由此可见函数中的this指的是全局对象。

二、作为对象的方法调用

当this作为对象的方法调用时this指向该对象。

三、作为构造函数调用

作为构造函数调用时，this指向new出的对象。

四、通过使用apply或call等方法后

apply()是函数对象的一个方法，
它的作用是改变函数的调用对象，
它的第一个参数就表示改变后的调用这个函数的对象。
因此当使用apply或call改变调用对象后this指的就是这第一个参数。
